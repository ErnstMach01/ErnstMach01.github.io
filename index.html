Reaction Tester
Reaction Tester
Created a light sequencer to test a player's reaction speed using a PIC16F690 Microchip. Written in assembly language with MPLAB X IDE. The user interface of this light sequencer consists of 8 LEDs and a button. The user would press the button to indicate the start of the game. After a randomly generated time delay, the lights would then start to turn on quickly one by one. The user simply needs to press the button again once he sees the LEDs starting to light up in sequence. If the player reacts before the fourth LED lights up, the LEDs then display a set of cheerful patterns.

3D Multiplexing Light Cube
3D Multiplexing Light Cube
Soldered together 27 LEDs together to create a 3 by 3 Led cube. All the LEDs aligned in a vertical column share a common anode (+). All the LEDs on a horizontal layer share a common cathode (-). Since trying to control each LED individually is an impractical idea, the cube was split into 3 layers. Each layer would have a pattern and then the program would then light up one layer at a time, but each layer would be lit up so quickly that the human eye would not be able to differentiate them - Multiplexing. Through multiplexing, the number of signals required to control the LEDs decreased from 27 lines to 12 lines. Created the program in C to send signals through a parallel port to turn on the LEDs along with a simple GUI to change light patterns.

PLC Sorting Machine
PLC Sorting Machine
Designed and machined a simple conveyor-belt sorting machine to sort 4 different types of objects. A metal disk, a black disk, a white disk, and a white disk with a hole in the center. The system uses 3 different sensors a metal sensor for the metal disc, a colour sensor for the black disc, and a pressure sensor for the white disk with a hole. The entire system was controlled by an industrial PLC and relays, taking input from sensors and sending signals to pneumatic pistons. When each sensor is activated a piston would trigger and push the respective disk off the belt into separate storage containers. The white disk would not trigger any of the sensors and would simply fall off at the very end of the belt. The entire system is controlled by an industrial PLC and relays.
